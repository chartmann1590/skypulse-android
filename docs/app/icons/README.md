# PWA Icons

Generate these icons from the SkyPulse radar logo SVG at:
stitch_skypulse_android_ui_design/skypulse_radar_logo/code.html

Required sizes:
- icon-192.png (192×192)
- icon-512.png (512×512)
- icon-maskable-192.png (192×192, with safe-zone padding for adaptive icons)
- apple-touch-icon.png (180×180)

Until generated, copy a placeholder PNG.
